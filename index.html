Dylan Nelson's portfolio for Unity: Create with Code

Unit 1: Prototype 1

Instructions: use the W & S keys to move forward and backward, use the A & D keys to turn left or right

This unit focused on learning the basic functions of the Unity editor, as well as basic C# scripts and rigid bodies. A character controller script was created that allows the user to use the WASD keys to move the truck forward or turn side to side. A camera follower script was created using the LateUpdate function to create a smooth following of the Van in the scene. Rigid bodies are used to allow collision between the van and the obstacles on the road.
